# Tailwind CSS Example Plugins

To get started, clone the project and install the dependencies:

```
# Using npm
npm install

# Using Yarn
yarn
```

After that, you can process the CSS by running:

```
# Using npm
npm run build

# Using Yarn
yarn run build
```

Load up `./index.html` in your browser and you're off to the races!
